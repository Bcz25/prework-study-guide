HTML

  • The HEAD element contains information about the webpage.
  • The BODY element represents the visible content shown to the user.
  • The META element within the header represent information readabele by the browser.
  • The P element represents a paragraph or block of text. You'll use this element extensively to wrap most of the text on your webpages.
  • The IMG element contains information about images that are displayed on the webpage.
  • The BR element creates a line of empty space

GIT

  • git: prefaces all commands when working with git
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git pull: used to fetch and download content from a remote repository and immediately update the local repository to match that content.
  • git push: used to upload local repository content to a remote repository.
